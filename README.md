TaskFlow – Task Management API

TaskFlow is a task management backend application built using Django and Django REST Framework.
It provides RESTful APIs for creating, updating, retrieving, and deleting tasks with secure user-based access control.

---

Tech Stack

Python
Django
Django REST Framework
PostgreSQL
JWT Authentication
Render (Deployment)

---

Features

User authentication using JWT
CRUD APIs for task management
User-specific task access
Filtering tasks by status and priority
Pagination support
Simple dashboard interface
